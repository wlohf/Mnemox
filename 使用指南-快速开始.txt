===============================================================
  一、快速启动（3步）
===============================================================

1. 一键启动所有服务
   双击运行: start_with_anythingllm.bat
   
2. 等待约 30 秒，直到所有服务启动完成

3. 测试是否成功
   打开新的命令行窗口：
   cd backend
   python test_integration.py
   
   看到 "✅ 所有测试通过！" 即可开始使用

===============================================================

🌐 前端界面:        http://localhost:5173
📚 API 文档:        http://localhost:8000/docs
🤖 AnythingLLM:     http://localhost:3001

  三、快速体验（推荐）

     -F "sync_to_anythingllm=true"

===============================================================
  五、配置说明
===============================================================

首次使用需要配置 backend/.env 文件：

1. 复制模板文件：
   copy backend\env.example.anythingllm backend\.env

2. 最小配置（必需）：
   ANYTHINGLLM_ENABLED=true
   ANYTHINGLLM_BASE_URL=http://localhost:3001
   DATABASE_URL=sqlite+aiosqlite:///./data/study.db

3. 配置 LLM（二选一）：
   
   选项A - 使用 OpenAI（需要 API Key）:
   OPENAI_API_KEY=sk-your-key-here
   
   选项B - 使用本地 Ollama（免费，推荐）:
   # 1. 安装 Ollama: https://ollama.ai
   # 2. 运行: ollama pull llama2
   # 3. 在 http://localhost:3001 配置使用 Ollama

===============================================================
  七、常见问题
===============================================================

Q1: 提示 "AnythingLLM 服务未启动"
A:  确保运行了 start_with_anythingllm.bat
    或手动启动 server 和 collector 服务

Q2: 上传文件后无法提问
A:  等待 10-30 秒让文档处理完成
    可以在 http://localhost:3001 查看处理状态

Q3: RAG 回答不相关
A:  确保问题与上传的文档内容相关
    尝试更具体的问题表述

Q4: 端口被占用
A:  修改 .env 中的端口配置
    或关闭占用端口的其他程序

===============================================================
  九、使用示例
===============================================================

Python 脚本示例:
```python
import requests

# 上传资料
files = {'file': open('my_doc.pdf', 'rb')}
data = {'title': 'Python教程', 'sync_to_anythingllm': 'true'}
r = requests.post('http://localhost:8000/api/materials/upload', 
                  files=files, data=data)
material_id = r.json()['id']

# 等待处理
import time
time.sleep(10)

# 提问
r = requests.post(f'http://localhost:8000/api/materials/{material_id}/ask',
                  json={'question': 'Python如何定义函数？'})
print(r.json()['answer'])
```
