# Mnemox v1.0.9

发布时间：2026-06-07

## 本次重点更新

### 1. 桌面更新改为直接下载并安装

- 设置页检测到新版本后，点击更新按钮会直接下载 Windows 安装包。
- 下载完成后会自动启动安装程序，不再跳转到 GitHub Release 页面让用户手动下载。
- 当 Electron 原生更新器不可用或没有正确返回可下载状态时，会使用更新清单中的 `downloads.windows` 作为兜底安装包地址。

### 2. 更新下载状态更清晰

- 下载过程中按钮显示为“正在下载更新”，并禁用重复点击。
- 桌面更新状态继续展示下载进度。
- 下载失败时会清理临时文件，并在界面上显示错误提示。

### 3. 安全与兼容性

- 兜底安装包下载仅接受 HTTPS 地址。
- 安装包文件名会从 Release 资产 URL 解析，并过滤不安全路径字符。
- 启动安装程序前会先停止内置后端进程，避免安装期间文件占用。

## 回归验证

已完成以下验证：

  - `npm test`

## 对用户的直接影响

- 从旧版本检查到新版本后，点击“下载并安装更新”即可进入下载和安装流程。
- 不再需要打开 GitHub 页面，也不需要手动寻找 `.exe` 安装包。
