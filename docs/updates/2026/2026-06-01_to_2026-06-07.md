# 更新记录：2026-06-01 至 2026-06-07

## 本周期目标

- 修正番茄工作法被设置面板意外打断的问题。
- 为番茄工作法补一个独立页面，提升桌面端展示体验。
- 把计划页和笔记页调整为更接近 Obsidian 的文档工作台。
- 让右侧边栏“今日任务”直接展示具体任务内容，而不是过于概括的摘要。
- 补齐对应的回归测试和页面自测。

## 已完成

### 1. 功能 / 体验

- 番茄工作法设置保存时不再打断已开始的专注计时，避免用户仅调整设置就丢失当前状态。
- 新增独立的番茄工作法页面，提供更完整的专注视图、任务展示和左上角激励语句。
- 左侧导航的番茄入口改为跳转独立页面，避免计时入口和设置入口混在侧栏卡片里。
- 右侧边栏“今日任务”改为直接展示当天具体任务，并支持跳转到当天计划文档。
- 计划页从弹窗输入改为中间工作区文档编辑，用户可以直接围绕当天计划进行连续编辑。
- 笔记页调整为更接近 Obsidian 的三栏结构，按文件夹与笔记文件组织内容。

### 2. 修复 / 稳定性

- 修复停止专注计时且不进入休息时，`currentTask` 未被清空的问题。
- 修复切换到空文件夹时，笔记编辑区仍残留上一条笔记内容的问题。

### 3. 测试 / 验证

- 为计划页文档工作台补充测试，确认不再走弹窗输入流程。
- 为笔记页文件夹切换补充测试，确认空目录时会清空当前笔记。
- 为番茄状态补充测试，确认停止时任务状态同步清理。
- 新增浏览器级自测脚本，覆盖番茄页、计划页、笔记页和右侧任务跳转链路。
