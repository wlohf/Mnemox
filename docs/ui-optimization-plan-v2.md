# UI 和功能优化方案 V2（基于用户反馈）

## 用户需求确认

### ✅ 保留的功能
1. **项目功能** - 资料和对话可以属于同一个项目，AI 查找更快
2. **番茄钟** - 重要！数据用于 AI 学习用户习惯，提供个性化服务
3. **日历计划** - 重要！用来记录当天要做的事情
4. **最常用功能** - 日历笔记 + 番茄钟

### ✨ 新增需求
- **鼓励话语框** - 显示励志话语，可以自己输入或让 AI 生成

### 📝 调整的流程
- **资料上传** - 先上传后分类（不强制选择项目）

---

## 优化方案（调整版）

### 核心思路
- **保留核心功能的可见性**（番茄钟、日历、笔记）
- **简化次要功能的入口**（错题本、复习中心等）
- **优化信息层级**（减少视觉噪音，突出重点）
- **增强激励体验**（鼓励话语、进度反馈）

## 第二部分：功能逻辑优化

### 1. 项目功能优化

#### 1.1 项目定位明确

**定义：**
- 项目 = 学习主题（如"考研数学"、"Python 入门"）
- 每个项目有自己的资料库（但资料可以不属于项目）
- 每个项目有自己的对话历史

#### 1.2 项目-资料关联

**流程：**
```
资料上传：
1. 用户上传资料
2. 可选：立即分配到项目（也可以跳过）
3. 资料进入"未分类资料"列表

资料分类：
方式 1：拖拽资料到项目
方式 2：选中资料 → 右键 → 分配到项目
方式 3：批量选择 → 分配到项目

资料使用：
- 对话属于项目时，默认使用该项目的所有资料
- 用户可以临时添加/移除资料（不影响项目设置）
```

#### 1.3 项目设置优化

**项目设置界面：**

## 第三部分：新增功能

### 1. 今日激励（鼓励话语）

#### 1.1 功能设计

**位置：** 右侧边栏顶部（始终可见）

**UI 设计：**
```
┌─────────────────────────────────────┐
│ 💪 今日激励                          │
│ ───────────────────────────────────│
│                                      │
│   "每一次努力都是进步的阶梯！"       │
│   "坚持就是胜利，加油！"             │
│                                      │
│ [换一句] [自定义...] [AI 生成]      │
└─────────────────────────────────────┘
```

**功能：**
1. **预设话语库**（50+ 条励志语录）
2. **自定义输入**（用户可以添加自己的话语）
3. **AI 生成**（根据用户学习情况生成个性化鼓励）
4. **定时更换**（每天自动换一句）

#### 1.2 AI 生成逻辑

**输入：**
- 用户最近的学习数据（番茄钟、完成任务、学习时长）
- 用户的学习目标（项目名称、待办事项）
- 当前时间（早上、下午、晚上）

**输出示例：**
```
早上：
"新的一天开始了！今天的目标是完成高等数学第三章，加油！"

下午（学习中）：
"已经完成 3 个番茄钟了，继续保持！休息一下再战！"

晚上（完成目标）：
"太棒了！今天完成了所有计划，明天继续加油！"

晚上（未完成）：
"今天虽然没完成所有任务，但进步就是胜利！明天继续努力！"
```

#### 1.3 数据库设计

**新增表：**
```sql
CREATE TABLE motivation_quotes (
    id INTEGER PRIMARY KEY,
    user_id INTEGER NOT NULL,
    content TEXT NOT NULL,
    source VARCHAR(20) DEFAULT 'preset',  -- preset, custom, ai_generated
    is_active BOOLEAN DEFAULT TRUE,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE daily_motivation (
    id INTEGER PRIMARY KEY,
    user_id INTEGER NOT NULL,
    date DATE NOT NULL,
    quote_id INTEGER,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (quote_id) REFERENCES motivation_quotes(id)
);
```

#### 1.4 API 设计

**端点：**
```python
# 获取今日激励
GET /api/motivation/today
Response: {
    "content": "每一次努力都是进步的阶梯！",
    "source": "preset"
}

# 换一句
POST /api/motivation/refresh
Response: {
    "content": "坚持就是胜利，加油！",
    "source": "preset"
}

# 自定义添加
POST /api/motivation/custom
Body: {
    "content": "我一定能考上研究生！"
}

# AI 生成
POST /api/motivation/generate
Response: {
    "content": "已经完成 3 个番茄钟了，继续保持！",
    "source": "ai_generated"
}

# 获取所有话语
GET /api/motivation/list
Response: [
    {"id": 1, "content": "...", "source": "preset"},
    {"id": 2, "content": "...", "source": "custom"}
]

# 删除自定义话语
DELETE /api/motivation/{id}
```
