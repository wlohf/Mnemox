# 🎨 界面更新 V2 - Obsidian 风格完整版

## ✨ 新增功能

### 1. 📅 右上角：可折叠日历

#### 折叠状态
- 显示"点击展开查看完整日历"提示
- 节省空间，保持界面简洁
- 点击 ▼ 按钮展开

#### 展开状态
- 显示完整月历
- 有计划的日期显示**绿色徽章**标记
- 点击任意日期打开计划编辑器
- 点击 ▲ 按钮收起

#### 计划功能
- **点击日期** → 打开计划编辑弹窗
- **输入计划** → 支持多行文本
- **保存计划** → 自动保存到该日期
- **查看计划** → 再次点击日期可查看/编辑
- **计划标记** → 有计划的日期显示绿色圆点

**示例使用场景**：
```
2024-01-11 的计划：
- 复习第一章
- 完成10道练习题
- 整理笔记
- 番茄钟：4个
```

---

### 2. 📁 左上角：资料文件列表

#### 显示格式
- **文件图标** + **文件名**（如：高等数学.pdf）
- 显示上传日期
- 删除按钮

#### 功能特点
- 点击文件可以选择学习
- 鼠标悬停显示完整信息
- 支持删除操作

**支持的文件格式**：
- PDF (.pdf)
- Word (.doc, .docx)
- 文本 (.txt)
- Markdown (.md)

---

### 3. ⬆️ 左下角：导入资料按钮

#### 位置
- 在资料列表下方
- 在错题本上方
- 大按钮，易于点击

#### 操作流程
1. 点击"导入资料"按钮
2. 选择文件（支持 PDF/Word/TXT/Markdown）
3. 文件自动添加到左上角资料列表
4. 显示成功提示

## 🎮 操作指南

### 如何使用日历功能

1. **展开日历**
   - 找到右上角"日历"卡片
   - 点击标题右侧的 ▼ 按钮
   - 日历展开显示完整月历

2. **添加计划**
   - 点击任意日期
   - 在弹窗中输入计划内容
   - 点击"保存"按钮
   - 该日期会显示绿色圆点标记

3. **查看计划**
   - 点击有绿点标记的日期
   - 查看已保存的计划
   - 可以编辑或添加新内容

4. **收起日历**
   - 点击 ▲ 按钮
   - 日历收起，节省空间

---

### 如何导入资料

1. **点击导入按钮**
   - 在左侧栏下方找到"导入资料"按钮
   - 点击按钮

2. **选择文件**
   - 在弹出的文件选择器中选择文件
   - 支持：PDF、Word、TXT、Markdown

3. **确认上传**
   - 文件自动添加到资料列表
   - 显示文件名和上传日期
   - 显示成功提示

4. **管理资料**
   - 点击文件名查看/学习
   - 点击 [×] 按钮删除

## 💡 使用技巧

1. **日历管理**
   - 不常用时收起日历，节省空间
   - 需要时展开查看本月计划
   - 绿点一目了然，哪天有安排

2. **资料整理**
   - 建议用清晰的文件名（如：高等数学-第一章.pdf）
   - 及时删除不需要的资料
   - 按学习顺序上传

3. **高效学习**
   - 早上查看今日计划
   - 学习时开启番茄钟
   - 晚上记录学习总结

## ❓ 常见问题

### Q1: 日历展开后看不到完整内容？
**A**: 可能是窗口太小，建议：
- 将右侧栏宽度调整到 350px
- 或者收起日历，需要时再展开

### Q2: 上传文件没反应？
**A**: 
- 检查文件格式（只支持 PDF/Word/TXT/MD）
- 刷新浏览器后重试
- 查看浏览器控制台是否有错误

### Q3: 计划保存后找不到？
**A**: 
- 确认点击了"保存"按钮
- 检查日期是否有绿色圆点
- 再次点击该日期查看
