## 🚀 如何开始使用

### 方式一: 快速体验 (推荐)

```bash
# 1. 进入项目目录
cd StudyAssitant

# 2. 一键启动所有服务
.\start_with_anythingllm.bat

# 3. 等待所有服务启动 (约30秒)

# 4. 运行测试验证
cd backend
python test_integration.py

# 5. 运行功能演示

```

### 方式二: 手动启动

```bash
# 终端 1: AnythingLLM Server
cd tools/anything-llm/server
yarn install  # 首次运行
yarn dev

# 终端 2: AnythingLLM Collector
cd tools/anything-llm/collector
yarn install  # 首次运行
yarn dev

# 终端 3: StudyAssistant Backend
cd StudyAssitant/backend
pip install -r requirements.txt  # 首次运行
# 配置 .env 文件
python -m uvicorn app.main:app --reload

# 终端 4: StudyAssistant Frontend (可选)
cd StudyAssitant/frontend
npm install  # 首次运行
npm run dev
```

## 📖 使用示例

### 示例 1: 上传并分析学习资料

```python
import requests
import time

# 1. 上传资料
files = {'file': open('python_tutorial.pdf', 'rb')}
data = {'title': 'Python教程', 'sync_to_anythingllm': 'true'}
response = requests.post('http://localhost:8000/api/materials/upload', 
                        files=files, data=data)
material_id = response.json()['id']

# 2. 等待处理
time.sleep(10)

# 3. 分析资料
response = requests.post(f'http://localhost:8000/api/materials/{material_id}/analyze')
print(response.json())
```

### 示例 2: 智能问答

```python
import requests

# 向 AI 提问
response = requests.post('http://localhost:8000/api/materials/1/ask',
    json={'question': 'Python中如何定义函数？'})
    
print(f"问题: {response.json()['question']}")
print(f"答案: {response.json()['answer']}")
```

### 示例 3: 生成学习大纲

```python
import requests

response = requests.post('http://localhost:8000/api/materials/1/generate-outline')
print(response.json()['outline'])
```

## 🔧 配置说明

### 必需配置

在 `StudyAssitant/backend/.env` 中配置:

```env
# 启用 AnythingLLM
ANYTHINGLLM_ENABLED=true
ANYTHINGLLM_BASE_URL=http://localhost:3001
ANYTHINGLLM_WORKSPACE=study-materials

# 数据库
DATABASE_URL=sqlite+aiosqlite:///./data/study.db
```

### 可选配置

```env
# 如果使用 OpenAI
DEFAULT_AI_PROVIDER=openai
OPENAI_API_KEY=sk-your-key-here

# 或使用本地 Ollama (推荐, 免费)
# 1. 安装 Ollama: https://ollama.ai
# 2. 运行: ollama pull llama2
# 3. 在 AnythingLLM Web 界面选择 Ollama
```

参考: `backend/env.example.anythingllm`

## ⚙️ 核心功能说明

### 1. 自动文档同步

- 用户上传资料 → 自动同步到 AnythingLLM
- 支持 PDF, TXT, MD, DOCX 等格式
- 可选择是否同步 (`sync_to_anythingllm` 参数)

### 2. RAG 智能问答

- 基于向量搜索的语义理解
- 回答基于用户上传的资料
- 支持引用溯源

### 3. 资料智能分析

自动回答三个问题:
- 资料的主要内容是什么？
- 有哪些重点知识点？
- 学习大纲是什么？

### 4. 学习大纲生成

AI 自动为资料生成:
- 章节划分
- 知识点梳理
- 学习路径建议

```bash

```

### 常见问题

1. **服务未启动**
   - 检查 3001 端口 (AnythingLLM Server)
   - 检查 8888 端口 (Collector)
   - 查看终端错误日志

2. **上传失败**
   - 确保 Collector 服务在运行
   - 检查文件格式是否支持
   - 查看 `data/uploads` 目录权限

3. **RAG 无响应**
   - 等待文档处理完成
   - 在 AnythingLLM Web 界面检查文档状态
   - 确认已配置 LLM 提供商

## 💡 使用技巧

### 1. 使用本地 LLM 节省成本

```bash
# 安装 Ollama
# Windows: https://ollama.ai/download
# Mac: brew install ollama
# Linux: curl -fsSL https://ollama.ai/install.sh | sh

# 下载模型
ollama pull llama2        # 基础模型
ollama pull llama2:13b    # 更强大的模型
ollama pull mistral       # 或其他模型

# 在 AnythingLLM Web 界面配置使用 Ollama
```

### 2. 优化问答质量

- ✅ 提问要具体明确
- ✅ 一次只问一个问题
- ✅ 包含相关上下文
- ❌ 避免模糊的问题

### 3. 管理多个学习资料

```python
# 为不同科目创建不同工作空间
from app.ai.anythingllm_provider import AnythingLLMProvider

provider = AnythingLLMProvider()
await provider.ensure_workspace("math-materials")
await provider.ensure_workspace("programming-materials")
await provider.ensure_workspace("english-materials")
```
