# Product

## Register

product

## Users

Mnemox is for learners and self-managers who move continuously between source materials, goals, notes, review, focused work, and reflection. They use the product while studying, planning the next session, recovering from stalled progress, or asking the Agent to make sense of scattered evidence.

## Product Purpose

Mnemox helps the Agent organize the next learning step from traceable evidence instead of acting like a generic chat assistant. Success means the user can see the current goal, the evidence behind recommendations, what will be written only after confirmation, and which memories or preferences are shaping the Agent's behavior.

## Brand Personality

Calm, evidence-led, capable. The interface should feel like a focused study cockpit: quiet enough for repeated daily use, precise about provenance, and confident without turning learning into noisy gamification.

## Anti-references

Avoid marketing hero layouts, decorative card grids, generic AI chatbot dashboards, noisy gamified streak surfaces, oversized illustrative panels, unexplained automation, and any interface that hides why the Agent is recommending or writing something.

## Design Principles

- Goal-first: the current learning goal and the next useful action should be easier to find than generic chat controls.
- Evidence is visible: notes, materials, events, and memory sources should remain traceable at the point of use.
- Confirmation before writes: Agent-generated tasks, notes, plans, and sensitive memories must be drafts until the user approves them.
- Dense but scannable: support serious study workflows with compact hierarchy, predictable controls, and clear grouping.
- Low-noise autonomy: automation should reduce cognitive load, not create another inbox of suggestions.

## Accessibility & Inclusion

Target WCAG AA contrast, keyboard-reachable controls, readable compact layouts on narrow screens, clear focus states, reduced-motion-friendly interactions, and labels that remain understandable without relying on color alone.
