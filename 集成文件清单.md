## 🔄 升级路径

如果您是从旧版本升级，需要：

1. **更新代码**
   ```bash
   # 所有新文件已创建，现有文件已更新
   ```

2. **安装新依赖**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

3. **配置环境变量**
   ```bash
   # 创建或更新 backend/.env
   copy env.example.anythingllm .env
   # 然后编辑 .env 文件
   ```

4. **启动 AnythingLLM**
   ```bash
   cd tools/anything-llm/server
   yarn install && yarn dev
   
   # 新终端
   cd tools/anything-llm/collector
   yarn install && yarn dev
   ```

5. **测试集成**
   ```bash
   cd backend
   python test_integration.py
   ```
