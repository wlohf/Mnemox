# Mnemox v1.0.7

发布时间：2026-06-06

## 本次重点更新

### 1. 联网搜索兼容性

- 官方 OpenAI 继续使用 Responses API 内置 Web Search。
- OpenAI-compatible 中转模型新增 Chat Completions tools 路径，可调用 Mnemox 本地 `web_search` 工具。
- 工具调用不兼容时自动降级到应用层网页搜索，并通过 SSE 返回搜索结果事件。

### 2. AI 设置与错误提示

- 供应商模型搜索和验证流程的错误提示更清晰。
- 聊天报错补充 AI 供应商设置指引，明确联网搜索复用当前供应商 API Key，不需要单独配置搜索 Key。
- 流式聊天继续保留网页搜索结果、搜索提示、记忆指标和学习反馈事件回调。

### 3. 番茄专注工作流

- 番茄页新增可复用专注任务、待办集、任务卡片、任务详情和任务级统计。
- 支持任务默认时长、任务卡背景色、完成状态切换和历史专注记录查看。
- 专注页布局扩展为左侧任务管理与中间计时工作台，更适合长期学习任务复用。

### 4. 右侧栏与今日任务

- 右侧栏布局偏好独立封装，统一同步卡片顺序、显示项、宽度、折叠状态和日历展开状态。
- 新增右侧栏布局持久化单元测试，覆盖桌面偏好与本地存储的合并、写入和规范化。
- 今日任务可在右侧栏直接勾选完成；目标任务完成时后端会记录 `completed_at`，恢复待办时清空完成时间。

### 5. 文档与发布元数据

- 应用版本、桌面包版本、前端包版本、更新清单和发布脚本默认版本同步到 `1.0.7`。
- README 和发布说明同步记录本次联网搜索、专注任务与布局持久化更新。

## 对用户的直接影响

- 使用 OpenAI-compatible 中转时，联网搜索可用性更高；不支持工具调用的模型会自动回退。
- AI 配置或联网搜索出错时，提示更接近可操作的设置指引。
- 番茄钟从一次性计时升级为可复用任务工作台。
- 右侧栏自定义布局更稳定，今日任务完成状态可以直接同步。
